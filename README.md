# Calplate - A Simple Web Calculator

Calplate is a simple calculator that can perform basic arithmetic operations like addition, subtraction, multiplication, and division.


## Features
* Standard arithmetic operations (+, -, *, /)
* Clear button (C) to reset the input
* Clean and simple user interface

## Technologies Used
* HTML5
* CSS3
* JavaScript

## How to Run
1.  Clone the repository: `git clone https://github.com/Mohithanjan23/calplate.git`
2.  Navigate to the project directory: `cd calplate`
3.  Open the `cal.html` file in your web browser.
