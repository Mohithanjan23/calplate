# Cal AI - AI-Powered Nutrition Tracker 🤖

**Cal AI** is a modern, minimalist nutrition application designed to make calorie tracking effortless. It leverages a conversational AI interface to log meals, provides a clean dashboard to track daily progress, and helps users achieve their health goals.

Visit the live site: [https://calplate.netlify.app/](https://calplate.netlify.app/)

---

## ✨ Core Features

* **🤖 Conversational AI Logging:** Log meals and activities simply by talking to the AI.
* **📷 AI Meal Scanner:** (Integrated into the conversational AI) Upload a photo to have the AI identify and log your food.
* **📊 Clean & Simple Dashboard:** A minimalist interface focused on your daily calories, macros, and recent activity.
* **🎯 Goal Setting & Onboarding:** A guided, multi-step process for new users to set up their personalized health and fitness goals.
* **🔒 Secure User Accounts:** Full user authentication and data privacy managed by Supabase.

---

## 🛠️ Tech Stack

* **Frontend:** React (with Vite)
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Routing:** React Router
* **Backend & Database:** Supabase
* **Deployment:** Netlify

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

1.  **Clone the repo**
    ```sh
    git clone [https://github.com/Mohithanjan23/calplate.git](https://github.com/Mohithanjan23/calplate.git)
    ```
2.  **Navigate to the project directory**
    ```sh
    cd calplate
    ```
3.  **Install NPM packages**
    ```sh
    npm install
    ```
4.  **Set up environment variables**
    * Create a file named `.env.local` in the root of your project.
    * Add your Supabase and other API credentials to this file.
5.  **Run the development server**
    ```sh
    npm run dev
    ```
